# FlowDrawing

Use [Wolkflow](https://github.com/geekq/workflow "Wolkflow") to draw flow chart directly.

## Usage

```
  git clone
```

To install GraphViz

__MAC OS:__

```
brew install GraphViz
```

__Ubuntu:__

```
sudo apt-get install graphviz
```

In FlowDrawing root path:

```
bundle install

unicorn
```

打开浏览器，在localhost:8080使用
